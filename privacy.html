Privacy Policy

Code & Coin Studio · Effective July 2, 2026

1. Who we are

Code & Coin Studio is a personal desktop application operated by the owner of the Code & Coin channel. It publishes the Operator's own videos to the Operator's own social media accounts.

2. What data the Studio handles

  • The Operator's own account credentials. OAuth access tokens for the Operator's own YouTube and TikTok accounts, plus basic profile information (such as display name) returned by those platforms during login.
  • The Operator's own content. Videos, titles, and descriptions created by the Operator for publishing.

3. What the Studio does not do

  • It does not collect, store, or process personal data of any third party or of any other platform user.
  • It does not sell, share, or transfer data to anyone.
  • It does not use tracking, analytics, or advertising technologies.

4. Storage and security

All data — including OAuth tokens — is stored locally on the Operator's own computer and is never transmitted anywhere except directly to the respective platform's official API (YouTube Data API, TikTok Content Posting API) over HTTPS.

5. Data retention and deletion

Tokens remain on the Operator's machine until revoked or deleted. Access can be revoked at any time from the platform's own security settings (TikTok: Settings → Security → Apps; Google: myaccount.google.com/permissions).
